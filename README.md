# Beacon - Interfaz Comercial

Esta es la interfaz comercial y landing page de **Beacon**, una aplicación diseñada para ofrecer un entorno sonoro estructurado basado en la resonancia armónica.

## Descripción

Beacon utiliza principios de armónicos naturales para crear espacios de coherencia y presencia. Esta interfaz sirve como punto de entrada para los usuarios, presentando la filosofía, la teoría detrás de la resonancia armónica y los enlaces para descargar la aplicación móvil.

## Características Principales

*   **Animación de Estados Mentales**: Una representación visual de cómo la presencia disuelve estados como la ansiedad o el ruido mental.
*   **Modos Visuales**: Adaptabilidad entre modos "Sun" (Día) y "Moon" (Noche/Espacio), reflejando diferentes estados energéticos.
*   **Información Teórica**: Explicación accesible sobre la teoría armónica resonante.
*   **Call to Action**: Enlaces directos a las tiendas de aplicaciones (App Store y Google Play).

## Tecnologías

*   React
*   Vite
*   Tailwind CSS
*   Framer Motion (para animaciones fluidas)

## Instalación y Desarrollo

1.  Clonar el repositorio.
2.  Instalar dependencias: `npm install`
3.  Correr el servidor de desarrollo: `npm run dev`
